# Super Store Sales Dashboard | Power BI

## Project Overview

This project presents an interactive Power BI dashboard built to analyze Super Store sales performance. The dashboard provides insights into sales, profit, orders, customer segments, and regional performance to support data-driven business decisions.

## Objectives

* Analyze overall sales and profit performance.
* Identify top-performing products and categories.
* Track sales trends over time.
* Compare performance across regions and customer segments.
* Discover business opportunities through data visualization.

## Tools Used

* Power BI
* Microsoft Excel
* Data Visualization
* Data Cleaning & Transformation

## Key Performance Indicators (KPIs)

* Total Sales
* Total Profit
* Total Orders
* Profit Margin
* Sales by Category
* Sales by Region
* Monthly Sales Trend

## Dashboard Features

* Interactive filters and slicers
* Region-wise sales analysis
* Category and sub-category performance
* Customer segment analysis
* Profitability insights
* Time-series sales trends

## Key Insights

* Technology category generated the highest sales.
* Certain regions contributed significantly more profit than others.
* Seasonal sales trends helped identify peak business periods.
* A few sub-categories showed high sales but low profitability.
